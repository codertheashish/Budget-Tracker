# 💰 Expense Tracker Web Application

A simple and responsive Expense Tracker Web Application built using HTML, CSS, and JavaScript. Easily manage your income and expenses, track your current balance, and store transactions locally in your browser.

---

## ✨ Features

* 💵 Add Income Transactions
* 💸 Add Expense Transactions
* ✏️ Edit Existing Transactions
* 🗑️ Delete Transactions
* 📊 Automatic Balance Calculation
* 💾 Local Storage Support
* 💱 Custom Currency Symbol
* 🔴 Negative Balance Indicator
* 📱 Responsive User Interface
* ⚡ Fast and Lightweight

## 📋 Key Functionalities

### Add Transactions

* Enter transaction name
* Choose Income or Expense
* Enter amount
* Save transaction instantly

### Edit Transactions

* Modify existing transactions
* Update amount and transaction type

### Delete Transactions

* Remove unwanted transactions
* Balance updates automatically

### Balance Tracking

* Income increases balance
* Expenses decrease balance
* Negative balances highlighted visually

### Local Storage

* Saves all transactions in browser storage
* Data remains available after page refresh
